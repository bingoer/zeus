# zeus
- websocket连接火币服务器
- 提供本地websocket服务端
- K线作为历史数据写入mongoDB
- 获取okex最新USD汇率写入redis
- 支持http查询历史K线数据
- 支持配置文件订阅交易对
- 支持其它交易所扩展
